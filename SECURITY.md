# Security Policy

## Supported Versions

This project is currently alpha software. Security fixes are handled on the latest `main` branch.

## Sensitive Data

Do not include Cookie headers, access tokens, saved files from `~/.maimai-cli/`, raw API responses, private posts, comments, profile data, or screenshots in public issues.

If you need to report a bug, redact all identifiers and private content first.

## Responsible Use

This project does not support login bypasses, CAPTCHA solving, anti-bot bypasses, credential sharing, or unauthorized data access. Reports requesting those capabilities will not be accepted.
